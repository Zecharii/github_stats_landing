About This Project

This project was inspired by the need for a comprehensive tool to analyze GitHub user data efficiently. As developers, we often found ourselves needing quick insights into our or others' GitHub activity. This dashboard is a solution to that need, providing detailed analytics and visualizations in an easy-to-use format.

This is a Portfolio Project for Holberton School. Check out the GitHub repository for more details.
