Welcome to GitHub Stats

Unlock the power of your GitHub data with our comprehensive analytics and insights.

GitHub Stats is a web app that generates insightful analytics about GitHub users, no login required. Simply enter a GitHub username and get a detailed dashboard with user insights, repository details, and data visualizations.

Get Started Now!

Key Features

User Insights

Gain comprehensive insights about GitHub users, including repositories, commits, issues, and more. Understand your activity and contributions at a glance.

Repository Details

Get detailed information about each repository such as commit counts, issues, and primary languages used. A deep dive into the heart of a user's GitHub activity.

Data Visualizations

Experience your data through visually appealing bar graphs, heat maps, and other visualizations. See your GitHub activity like never before.

About GitHub Stats

GitHub Stats was built using Flask, matplotlib, pandas, MySQL, HTML, and CSS. It leverages the GitHub API to bring you detailed analytics and visualizations in an easy-to-use format.

This project was carried out by Zechariah Gaiya as part of the requirement for the ALX Software Engineering program. Inspired by the need for a comprehensive tool to analyze GitHub user data efficiently, this dashboard provides developers with quick insights into GitHub activity. Check out the GitHub repository for more details.
